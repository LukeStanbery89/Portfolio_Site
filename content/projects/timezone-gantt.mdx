# Timezone Gantt Chart

A practical web application for visualizing and comparing multiple timezones simultaneously in an intuitive gantt chart format.

## Features

- **Visual Timezone Comparison**: See multiple timezones side-by-side in a gantt chart layout
- **Interactive Interface**: Easily add, remove, and compare different time zones
- **Real-time Updates**: Current time displayed across all selected timezones
- **Clean UI**: Modern, responsive design for optimal viewing on any device

## Use Cases

Perfect for:
- Coordinating meetings across global teams
- Planning international calls and conferences
- Managing remote team schedules
- Travel planning and scheduling

## Technology Stack

Built with modern web technologies:
- **Next.js**: React framework for production
- **TypeScript**: Type-safe development
- **React**: Component-based UI
- **Responsive Design**: Mobile-friendly interface
